# Dictionaries Practice (Python)

This folder contains practice programs focused on understanding and applying **dictionaries in Python**.

## Topics Practiced

* Creating dictionaries
* Storing key-value pairs
* Checking if a key exists
* Updating values
* Traversing dictionary using loops
* Using dictionaries for frequency counting problems

## Programs Included

1. **frequency_counter.py**
   Takes multiple numbers as input, stores them in a list, and uses a dictionary to count the frequency of each number.

## Learning Outcome

This practice helps in building strong problem-solving skills using key-value data structures. Dictionaries are widely used in data processing, algorithm design (hashing concepts), and real-world applications such as counting, grouping, and fast lookup operations.
